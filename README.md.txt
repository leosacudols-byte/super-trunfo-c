# Super Trunfo em C 🎴

Este projeto simula o jogo **Super Trunfo** em C, com três níveis de complexidade:

## Etapa 1
- Comparação de um único atributo
- Uso de `if` e `if-else`

## Etapa 2
- Comparação de múltiplos atributos
- Uso de `if-else if` e `switch`

## Etapa 3
- Comparação de dois atributos ao mesmo tempo
- Uso de operador **ternário**
- Integração de todas as estruturas de decisão aprendidas

## Como compilar e executar
```bash
gcc etapa1.c -o etapa1
./etapa1

gcc etapa2.c -o etapa2
./etapa2

gcc etapa3.c -o etapa3
./etapa3
